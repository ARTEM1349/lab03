Завдання №13: Знайти максимальне просте число, що належить діапазонуь [a,b] 

Основна частина:
- опис роботи: Просте число - це число, яке має тільки два дільники це (1 і саме число). Потрібно вивести найбільше просте число з діапазону який введе користувач.

#include <stdio.h>

int main()
{
   int a , i, max = 2;
   scanf("%f",&a);
   for(int b = 2; a > b; b++)
 {      
     i = 2;
     while(b%i != 0)
       {
           i++;
       }
       if(b == i && b > max)
       {
          max = b;      
       }      
 }
      return 0;
}
